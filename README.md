## Run App

Đề chạy dự án cần làm các bước sau:

- git clone https://github.com/xuankhanh1612/FrontEndNFT.git

- Mở terminal và gõ lệnh

`npm install`

`npm start` 

- Mở thêm 1 terminal nữa để chạy file `server.js` bằng cách

`node server`
